# Atividade - PLN/NLP e Visão Computacional

## Exercício 1 - OCR de Comprovantes

### Técnicas utilizadas
- ROI (Região de Interesse)
- Leitura e manipulação de imagens
- OCR

### Bibliotecas utilizadas
- Pillow
- pytesseract

### Descrição
O sistema realiza a leitura automática de comprovantes de pagamento utilizando OCR.

O usuário digita um valor no terminal e o sistema procura automaticamente imagens contendo o valor informado.
